# Aviation MOE Compliance Checker

## What It Does
- Upload a PDF of your MOE
- Converts it to text
- Finds all the EASA codes mentioned
- Compares them to the official rules
- Uses AI to spot issues or risks
- Gives you a report with problems and where to find them
